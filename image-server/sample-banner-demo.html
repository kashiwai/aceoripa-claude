🔧 画像サーバー使用方法

画像生成サーバーが http://localhost:9015 で稼働中です。

API エンドポイント:

  • GET /sample-banners - 全サンプル画像の取得
  • POST /generate-banner - カスタムバナー生成

バナー生成例:

fetch('http://localhost:9015/generate-banner', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    type: 'card-pack',
    text: 'ポケモンカード151',
    width: 400,
    height: 600
  })
})

🎮 これらのバナーはオンラインオリパサイト「DOPA」のデザインを参考に作成されています
